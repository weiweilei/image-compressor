# 图片压缩工具

这是一个简单易用的在线图片压缩工具，具有苹果风格的界面设计。用户可以上传图片，预览压缩效果并下载压缩后的图片。

## 在线使用

访问：https://你的用户名.github.io/image-compressor

## 功能特点

- 支持PNG、JPG等常见图片格式
- 实时预览压缩前后的图片效果
- 显示压缩前后的文件大小对比
- 可调节压缩比例
- 支持压缩后图片下载
- 支持自定义目标文件大小
- 优雅的苹果风格界面设计

## 使用说明

1. 打开网站
2. 点击上传区域或将图片拖拽到上传区域
3. 输入期望的目标文件大小
4. 等待压缩完成
5. 点击下载按钮保存压缩后的图片

## 隐私说明

- 所有图片处理都在浏览器本地进行
- 不会上传图片到服务器
- 无需担心隐私泄露

## 技术栈

- HTML5
- CSS3
